# Locked Product Roadmap

This build locks the Todo app direction as a production-minded, mobile-first PWA using Laravel Blade plus a hybrid AJAX/fetch interaction model.

## Locked foundations
- Todo app only
- Mobile-first compact UI
- Brand color `#7e57c2`
- PWA direction
- In-app notifications
- Browser notification flow
- Role-permission architecture
- Admin dashboard add-on
- Smart task calendar direction
- Responsive app-like lists and details

## Locked implementation roadmap
### Phase 1
- Foundation hardening
- Dashboard data cleanup
- Branding/login cleanup
- Notification badge and UI cleanup

### Phase 2
- Database-backed RBAC
- Roles, permissions, pivots
- Admin access protection
- Notification registration and live feed support

### Phase 3
- Service-layer business logic separation
- Task engine cleanup
- Recurring/reminder cleanup
- Query/index performance pass

### Phase 4
- Full role management UI
- Permission assignment UX
- Safer admin controls

### Phase 5
- Advanced notification delivery
- Queue/background push path
- Reminder preferences refinement

### Phase 6
- Premium mobile UX polish
- Calendar refinement
- Offline/load/empty states polish

### Phase 7+
- Deployment hardening
- Testing
- Audit/logging
- Future team/workspace readiness
